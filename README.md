# Тестовий проєкт G&G

Це тестовий проєкт — приклад верстки.

## Команди для роботи з проєктом

- Встановити залежності: `npm install`
- Запустити в режимі розробки: `npm run dev`
- Збілдити проєкт: `npm run build`
- Відкрити збілджений проєкт для перегляду: `npm run preview`

## Особливості перевірки

Для коректної перевірки потрібно:

1. Спочатку виконати білд: `npm run build`
2. Потім запустити перегляд збілдженої версії: `npm run preview`
